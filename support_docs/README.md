# Support Documentation

This folder contains supplementary learning materials for the YesFundMe project.

## Folder Structure

```
support_docs/
├── cheatsheets/    # Quick reference guides for technologies used
├── slides/         # Presentation slides for teaching
└── pdfs/           # Printable documentation
```

## Cheatsheets

Quick reference guides covering:
- Tailwind CSS basics
- React patterns
- Express routes
- SQLite queries
- JWT authentication
- Git workflow
- Debugging tips

## Usage

These materials are designed to support students working on the YesFundMe project. Use them as reference while building features or fixing bugs.
